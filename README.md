# Building & Deploying a Full-Stack MERN App on Azure

This repository contains materials for the **Technical Workshop Series:  
Building and Deploying a Full-Stack MERN App on Azure with CI/CD**.

The workshop is an **8-week, hands-on series** where participants build and deploy a simple full-stack MERN application using modern development and deployment practices.

---

## Workshop Structure

- The workshop runs weekly and builds incrementally toward a complete, deployed application.
- Materials for each session (such as slides) are organized by **week-specific branches**.
- Code repositories for the frontend and backend are shared separately.

---

## Session Materials

- **Week 1:** Introduction & Backend Setup  
  Slides are available in the `week-1` branch.
- Backend Repo: https://github.com/singhlify/azure-mern-workshop-backend

Additional weeks and materials will be added as the series progresses.
